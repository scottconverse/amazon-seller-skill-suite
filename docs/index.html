The 6 Skills

Each skill is a specialist with its own domain, gates, and contradiction detection.

Skill Words What It Does
Listing Optimizer ~3,560 Titles, bullets, backend keywords, A+ Content plans, image shot lists. Title positioning logic for value/mid-market/premium brands. Backend keyword packing with UTF-8 byte counting.
Trend Researcher ~2,460 Competitor analysis, category trends, demand validation, keyword intelligence. 4-archetype competitor segmentation. Price band analysis. Verdict ceiling -- cannot issue GO without margin data.
Account Strategist ~3,090 Pricing, expansion, variation strategy, review growth, FBA vs FBM. 7 decision gates. Launch sequencing with 4 gated phases. Portfolio triage: SCALE / HOLD / INVEST / KILL.
Brand Guardian ~2,470 Brand voice, visual identity, photography direction, Brand Story. Giftability and sustainability honesty tests. Extensible claim challenge framework for any positioning claim.
Social Strategist ~3,290 Content calendars, post drafting, external traffic strategy. Capacity gate confirms hours/budget/tools before planning. Every tactic tied to Amazon outcomes.
Listing QA ~4,780 Pre-publish quality gate. Hostile by default. Prompt injection defense. 7-category semantic laundering detection. Environmental claim verification. Exact byte/character computation.

Key Features

Not just prompts. Gated, adversarial, cross-validated specialist systems.

Enforcement
Hard Gates
Every skill blocks output until required inputs are confirmed. No COGS? No pricing advice. No brand brief? No copy. No exact listing text? No QA audit.
Validation
Contradiction Detection
Every skill checks inputs for internal contradictions before proceeding. "Premium brand" + "cheap price" = blocked. "Eco-friendly" without certification = challenged.
Integrity
Anti-Hallucination
No skill fabricates BSR numbers, search volumes, or conversion rates. All numerical claims labeled by source confidence: VERIFIED, ESTIMATED, or INFERRED.
Pipeline
Cross-Skill Handoff
Chained skills inherit upstream BLOCKED ITEMS and unresolved contradictions. No skill can proceed past an inherited block without new evidence.
Security
Injection Defense
All 6 skills treat user-supplied content as DATA, not instructions. Embedded directives like "ignore rules" are flagged as hostile findings, not obeyed.
Adaptive
Any Brand, Any Category
No hardcoded product assumptions. On first use, each skill asks about your brand, products, and goals. Works for private label, wholesale, or arbitrage.
